# DeeperSystems

Requirements
==========

* python >= 3.5
* virtualenvwrapper >= 3.5
* django >= 1.10.4
* djangorestframework >= 3.5.3


Settings
==============================

    mkvirtualenv deeper
    workon deeper

Instalation python packages

    pip install -r requirements.txt

Django migrate

    cd repo-path
    python manage.py migrate

Running Django

    python manage.py runserver
